# **课程核心内容总结：汇川AutoShop软件入门**

## **一、软件简介与核心认知**
1.  **软件名称**：汇川AutoShop。
2.  **主要用途**：专用于汇川**小型PLC**（如H5U、PPT-A8系列）的编程、下载和调试。
3.  **软件定位**：界面和操作逻辑与西门子S7-200/SMART 200类似，直观易上手，适合初学者入门。
4.  **重要区别**：汇川中型PLC使用另一款软件，不要混淆。

## **二、软件安装要点**
-   **准备**：安装前务必**关闭杀毒软件和Windows防火墙**，防止安装文件被误删或拦截。
-   **来源**：从汇川官网或“优易课堂”获取安装包。
-   **路径**：安装时选择**英文路径**（如 `D:\AutoShop`），且确保磁盘空间充足，避免安装异常。
-   **步骤**：解压后运行 `Setup.exe`，语言选择“中文简体”，按提示操作，建议勾选“创建桌面图标”。

## **三、软件界面与核心功能区**
软件界面可分为四大核心区域，其功能概括如下表所示：

| 区域 | 主要功能 | 包含内容 |
| :--- | :--- | :--- |
| **左侧目录区** | **工程管理** | 系统/全局变量表、程序块（主程序、子程序等） |
| | **功能配置** | PLC扩展模块、运动控制轴、通讯参数等设置 |
| | **调试工具** | 变量监控表、交叉引用（查询变量使用位置） |
| **中间编程区** | **程序编写** | 梯形图编辑区，可从右侧指令库拖拽指令进行编程 |
| **底部输出区** | **信息反馈** | 显示编译错误、下载状态、通讯结果等关键信息 |
| **顶部菜单栏** | **全集操作** | 文件、编辑、编译、下载、调试、工具等所有功能 |

## **四、PLC编程基本工作流程**
对于初学者，应熟练掌握以下核心操作流程：
1.  **新建工程**：选择PLC型号（如PPT-A8）、编程语言（梯形图）。
2.  **编写程序**：在梯形图编辑区编写逻辑。
3.  **编译程序**：检查程序语法错误。
4.  **连接PLC**：通过通讯线将电脑与PLC连接。
5.  **下载程序**：将编译成功的程序下载到PLC中。
6.  **调试监控**：在线监控程序运行状态，排查问题。

---

## 五、**初学者应当重点掌握**

1.  **工程管理**：学会新建、保存工程，并正确选择PLC型号。
2.  **梯形图编程**：掌握基本指令（如触点、线圈、定时器）的添加和使用。
3.  **程序编译与下载**：理解编译的作用，掌握下载程序的完整步骤。
4.  **在线监控与调试**：学会使用变量监控表实时查看数据变化，这是调试程序的关键。
5.  **模拟器使用**：在无实物PLC时，使用软件自带的模拟器测试简单逻辑。

### **常用快捷键**

| 功能 | 预计快捷键 | 说明 |
| :--- | :--- | :--- |
| **保存工程** | `Ctrl + S` | 最重要的快捷键，经常保存防止丢失 |
| **编译程序** | `F7` 或 `Ctrl + F7` | 检查程序错误 |
| **下载程序** | `F8` 或 `Ctrl + F8` | 将程序写入PLC |
| **在线监控** | `F9` 或 `Ctrl + F9` | 连接PLC并查看程序运行状态 |
| **撤销/重做** | `Ctrl + Z` / `Ctrl + Y` | 编程必备 |
| **查找/替换** | `Ctrl + F` / `Ctrl + H` | 快速定位变量或指令 |
| **插入一行** | `Insert` | 在编程时快速插入新行 |
| **删除一行** | `Delete` | 删除选中的行或元件 |
| **常开触点** | `F5` | 快速输入一个常开触点 |
| **常闭触点** | `F6` | 快速输入一个常闭触点 |
| **线圈** | `F7` | 快速输入一个输出线圈 |

**提示**：最准确的快捷键列表在AutoShop软件的菜单栏中查看，每个菜单项的名称后都会标注其快捷键。
