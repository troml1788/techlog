# **《PLC输出单元类型与H5U接线》课程知识点详细总结**

本节课的核心是理解PLC如何控制外部设备，重点讲解了两种输出类型的原理、特点、应用场景，并以汇川H5U型号为例进行了具体的接线说明。

## **一、PLC输出单元的两种核心类型**

输出单元是PLC驱动外部负载（如灯、电机、继电器）的接口。

| 特性维度 | **继电器输出** | **晶体管输出** |
| :--- | :--- | :--- |
| **工作原理** | 通过内部微型继电器的**机械触点**的吸合与断开来控制电路。 | 通过半导体元件（晶体管）的**电子开关**特性来控制电路。 |
| **响应速度** | **慢**（约10毫秒）。由于机械部件有动作时间，不适合高频开关。 | **极快**（小于0.2毫秒）。无机械延迟，适合高速脉冲控制。 |
| **负载兼容性** | **强**。可连接交流（AC）或直流（DC）负载，对电压类型不敏感。 | **弱**。通常只能连接**直流（DC）** 负载（如常见的24V DC设备）。 |
| **带载能力** | **电流较大**（单个触点通常可达2A），可直接驱动一些功率较大的设备。 | **电流较小**（通常为0.5A或更小），一般用于驱动小功率设备或作为信号源。 |
| **寿命** | 有机械磨损，**寿命有限**（尤其在频繁开关时）。 | 无机械磨损，**寿命极长**。 |
| **核心应用场景** | 对速度要求不高的场合，如控制电灯、电磁阀、小型接触器等。 | **高速控制场景**，如控制伺服驱动器、步进驱动器（需要发脉冲进行定位）、高频闪烁灯等。 |

**关键结论**：在自动化学习中，**优先选择晶体管型PLC**（如H5U），因为它能更好地满足后续伺服/步进电机等精密控制的学习需求。若需要控制交流大负载，可通过PLC的输出来控制一个中间继电器，再由中间继电器去驱动大负载。

## **二、H5U PLC的输出配置与接线详解**

**1. 输出点分组与公共端**
*   H5U的输出点分为两组，**公共端**是接线的关键。
    *   **Y0 - Y7**：这8个点共用公共端 **C0**。
    *   **Y10 - Y13**：这4个点共用公共端 **C1**。
*   **重要规则**：C0和C1端子**必须接入电源的0V（负极）**。PLC的输出点（如Y0）在导通时，相当于提供一个到0V的路径。

**2. 晶体管输出接线示例（控制继电器）**
这是最常用的接线方式，将PLC的小电流信号转换为可控制大功率负载的电路。

*   **目标**：用PLC的Y0输出点控制一个中间继电器线圈，进而用继电器的触点去控制一个灯泡或电机。
*   **接线步骤**：
    1.  **电源准备**：准备一个24V DC电源。正极（24V+）备用，负极（0V）接到PLC的**C0**公共端。
    2.  **连接负载回路**：
        *   将24V DC正极接到**中间继电器线圈**的一个端子。
        *   将继电器线圈的另一个端子接到PLC的**Y0**输出点。
        *   **此时，当Y0被程序置为“ON”时，电流路径为：24V+ → 继电器线圈 → Y0 → C0 → 0V。回路接通，继电器吸合。**
    3.  **控制大负载**：将主电路（如220V AC灯泡）的火线接到**继电器的一个常开触点**，再从触点另一端接到灯泡，灯泡另一端接零线。当继电器吸合，其触点闭合，灯泡点亮。

**3. 接线细节与注意事项**
*   **极性继电器**：部分继电器线圈有正负极之分（通常带有并联的指示灯二极管）。
    *   **正确接法**：线圈正极接电源正极，线圈负极接PLC输出点。
    *   **现象**：如果接反，继电器可能仍会动作，但其**指示灯可能不会点亮**，不利于故障诊断。
*   **PLC输出的角色**：在NPN型晶体管输出（常见类型）中，PLC输出点相当于**开关信号的负极（Sink）**。外部电源的正极通过负载（如继电器线圈）接入，再连接到Y点。

## **三、其他要点与学习建议**

*   **灵活应用**：在实操中，可以根据个人习惯和现场情况选择最方便的接线方式，但必须遵循电气安全规范。
*   **状态监控**：继电器上的二极管指示灯是判断其是否得电的直观依据，在调试和排查故障时非常重要。
*   **软件匹配**：为PLC下载程序时，确保编程软件中的设备型号与实际使用的PLC型号（如H5U）匹配，并下载正确的工程版本。
