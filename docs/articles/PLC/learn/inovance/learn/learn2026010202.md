# **PLC实物接线核心知识点总结**

本次讲解系统性地介绍了PLC硬件接线的基本原理、操作规范以及针对不同器件的具体接线方法。


## **一、 基本原则与安全规范**

1.  **供电基础**：
    *   PLC工作电源为直流24V，由外部开关电源将交流220V转换而来。
    *   接线时必须严格遵守 **“断电操作”** 原则，确保安全。
    *   接通电源后，PLC数码管显示且RUN指示灯亮起，表示处于运行状态。下载程序需将拨码开关拨至STOP位置。

2.  **PLC输入点通用原理**：
    *   PLC的每个输入点（如X0, X1）内部结构类似一个“无极性灯泡”，核心是形成一个电流回路。
    *   **关键概念 - 公共端（S/S端）**：
        *   此端子是所有输入回路的公共端，相当于灯泡的零线端。
        *   **必须统一接线**：整个PLC的所有输入点，其S/S端只能统一接24V正极（+24V）或统一接负极（0V），不能混接。

## **二、 输入信号的两种基本接法**

根据S/S端所接的极性，可分为两种接法。对于普通开关，两种接法功能无异。

1.  **漏型输入（Sink，也称拉型输入）**
    *   **接法**：S/S端子接至**+24V**。
    *   **电流路径**：电流从0V出发 → 流经外部开关 → 流入PLC输入点（X点）→ 在PLC内部流向S/S端 → 最后回到+24V。
    *   **特点**：信号有效时，输入点相对于S/S端为低电平。

2.  **源型输入（Source，也称推型输入）**
    *   **接法**：S/S端子接至**0V**。
    *   **电流路径**：电流从+24V出发 → 流经外部开关 → 流入PLC输入点（X点）→ 在PLC内部流向S/S端 → 最后回到0V。
    *   **特点**：信号有效时，输入点相对于S/S端为高电平。

## **三、 各类传感器接线详解**

1.  **普通开关（按钮、拨码开关等）**
    *   **特点**：无极性，相当于一个简单的导线通断开关。
    *   **接线**：可灵活采用上述任一接法（漏型或源型），只需接入回路即可，性能无差异。

2.  **两线制传感器（如磁性开关）**
    *   **特点**：有极性！内部相当于一个带二极管的开关，电流只能从正极（棕色线）流向负极（蓝色线）。
    *   **接线**：
        *   必须注意极性，接反则无法工作。
        *   可接入漏型或源型回路，但需确保其电流方向正确。例如，在漏型接法中，其棕色线应接0V，蓝色线接X点。
    *   **应用**：常用于检测气缸活塞位置（伸出/缩回）。

3.  **三线制/四线制传感器（NPN/PNP型，如接近开关、光电开关）**
    *   **电源线通用接法**：
        *   **棕色线（Brown）**：接 **+24V**。
        *   **蓝色线（Blue）**：接 **0V**。
        *   **黑色线（Black）**：**信号输出线**，接PLC的输入点（X点）。
    *   **核心区别（内部开关位置）**：
        *   **PNP型**：信号开关位于**棕色线（+24V）和黑色信号线**之间。
            *   **工作状态**：传感器被触发时，黑色信号线输出+24V高电平。
        *   **NPN型**：信号开关位于**蓝色线（0V）和黑色信号线**之间。
            *   **工作状态**：传感器被触发时，黑色信号线输出0V低电平。
    *   **与PLC接法的匹配**：
        *   **PNP型传感器** 更适合与 **源型输入（S/S接0V）** 的PLC配合使用。
        *   **NPN型传感器** 更适合与 **漏型输入（S/S接+24V）** 的PLC配合使用。


## **四、 快速判别传感器的方法**

当传感器型号不明时，可通过以下方法判断是NPN型还是PNP型：

1.  **准备**：一个24V开关电源，一个万用表（调至直流电压档）。
2.  **接线**：将传感器的**棕色线接+24V**，**蓝色线接0V**。
3.  **测量**：**触发传感器**（如用金属物靠近接近开关），用万用表测量**黑色信号线**与**电源线**之间的电压。
    *   **若黑线与棕线（+24V）间电压接近24V** → 为 **PNP型**。
    *   **若黑线与蓝线（0V）间电压接近24V** → 为 **NPN型**。
    *   **关键**：测量时务必确保传感器处于被触发的状态。

## **五、核心要点回顾**

*   **安全第一**：断电接线。
*   **回路思维**：所有接线都是为了形成完整的电流回路。
*   **公共端统一**：S/S端极性必须统一。
*   **极性意识**：区分无极性器件（普通开关）和有极性器件（传感器）。
*   **NPN/PNP本质**：记住其内部开关在电源正极（PNP）还是负极（NPN）一侧，决定了信号输出的电平。
