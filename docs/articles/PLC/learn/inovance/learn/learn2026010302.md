# 抢答灯（含多地控制）
## 业务场景
![alt text](assets/learn2026010302/image.png)

游戏规则：
+ 主持人按下**开始抢答**按钮后，开始抢答
+ 小学生组按下任一一个按钮则小学生组抢答成功
+ 中学生组按下抢答按钮则中学生组抢答成功
+ 教授组必须同时按下抢答按钮才算抢答成功
+ 一轮结束主持人可以按**复位**准备下一轮
## I/O规划
|I/O规划|作用|
|-|-|
|X0|小学生组按钮1|
|X1|小学生组按钮2|
|X2|中学生组按钮|
|X3|教授组按钮1|
|X4|教授组按钮2|
|X5|主持人复位按钮|
|X6|开始抢答按钮|<-主要是防止有人持续按着
|-|-|
|Y0|小学生组抢答灯|
|Y1|中学生组抢答灯|
|Y2|教授组抢答灯|
|Y3|开始抢答灯|
