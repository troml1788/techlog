# 📖 为什么会想起读书？

---

::: tip 时光札记
若把 75 年的人生拆解成月份，每个月都算作一格小小的**时光柜**，人的一生也不过堪堪 **900 格**。

算到 2025年12月23日，我刚满 24 周岁，细数下来，手里还握着 **612 格** 未开启的时光。

我自知是平凡的大多数，却仍揣着一份想奔赴「成功人生」的执念。可前路漫漫，一切都藏在迷雾里，我更怕一腔热血付诸东流，怕拼尽全力后依旧没能抵达预想的彼岸，那样的落差，或许会狠狠挫伤心底的那股心气。

唯有**读书**，是一场稳赚不赔的奔赴。
从 2026 年 1 月 1 日起，若能每月读完一本书，这 612 格时光里，便能攒下 **612 本**的收获与力量。
:::
