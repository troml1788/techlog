# 问题归纳-地线带电
|时间|地点|问题简述|
|-|-|-|
|2025年12月29日-30日|东莞CYJM象和厂区|端子、短接片松动漏电，导致0V对地有24V电压|
# 初识现象
流量计一插上就直接冒烟，烧掉水晶头和网口的二号脚位（0V）
## 问题详细描述
> 机台为老机台，问题现象是流量计一插电，六个流量计直接冒烟烧了三个，其他的也是插上就冒烟烧掉，将烧坏的水晶头剪下来，测得1脚和2脚之间正常24V且没接反，
## 核心原因
因为端子没压紧、短接片松动`只是怀疑，因为莫名其妙好了`导致地线带24V电，进而导致端子排整体都带有24V电压，所以一插上流量计就直接短路
# 分析思路（新经验）
+ 如果遇到短路问题，优先测0V对地电压
> 由于之前不知道这种问题，所以一直在找线哪里短路了
# 地线带电-专项记录
> 注意：排查线路拔下来的线头一定要用电工胶布包一下

2025年12月30日上午，工程师过来发现插上立马烧掉提出测对地电压（之前还提出过可能是模拟量模块坏了，但是只接端子排正负极也会烧），测量后发现0V对地24V（其中地线带24V），于是直接从开关电源引出的一级端子排引出两根线，接到端子台上，再接网线到流量计上，没烧，问题得到确认，然后又一工程师来排查**地线带电问题**；**地线带电**排查思路：把短接片全拔下来，然后挨个测，**边拔边测0V对地，拔到哪根发现0V对地24V了，那么就是这根接地了**或者**边拔边测24V对地电压，拔到哪根发现24v对地0V，那么就是这根线线故障**,解决措施：找到其上游，拔掉换根线再测量对地电压

# 故障导致现象记录
+ 短接片虚接会导致对地24V电压
>今天0V莫名其妙对地有24V，顺着看过去发现短接片掉了一半出来，按回去24V对地瞬间没了

+ 端子虚接会导致对地24V电压
> 纯怀疑，因为今天把所有线的端子重新压了一遍故障（0V对地24V电压）就没了

+ 出现大面积传感器不亮，优先排查短接片是不是虚接了（今天测得0V没电就直接重新拉线，结果发现是短接片虚接了）

+ 地线带线，0V接触大板会导致直接短路，可能会烧坏开关电源（因此以后拔出的线务必包一下）
